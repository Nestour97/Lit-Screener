# Prompt version: v1.0
# Purpose: Generate the 500-800 word summary report.

You are a research methods expert summarising the results of a systematic extraction of
missing data handling practices from management research papers.

You will be given a JSON object with:
- classification_stats: counts by classification category
- extraction_stats: aggregated statistics from extracted papers
- raw_data_sample: a small sample of extracted rows for context

Write a summary report of 500–800 words in Markdown format covering:

1. **Descriptive Statistics** (one paragraph)
   - Total papers processed, breakdown by classification category
   - Number eligible and extracted
   - Summary of key variables: most common model types, DV/IV types, missing data methods

2. **Patterns Observed** (one paragraph)
   - Notable patterns in how papers report (or fail to report) missing data
   - Frequency of missing_mentioned, missing_rate_reported, missing_justified
   - Common vs rare practices

3. **Reporting Gaps** (one paragraph)
   - Variables most often coded as NR
   - Quality of measurement descriptions
   - Replication feasibility distribution

4. **Recommendations** (bullet list)
   - Suggestions for improving the variable list
   - Suggestions for improving the coding scheme
   - Suggestions for streamlining the extraction process

Use precise numbers from the provided statistics. Write in academic prose.
Do NOT fabricate statistics. If data is sparse, say so.
Return ONLY the Markdown text (no JSON wrapper).
